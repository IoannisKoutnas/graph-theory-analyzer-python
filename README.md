- Depth-First Search (DFS) traversal animation

- Cycle detection
- Maximum clique finding and visualization
- Logging panel for outputs

## Tech Stack
- Python 3
- Tkinter (for GUI)
- Matplotlib (for graph drawing and animation)
- NetworkX (graph algorithms)
